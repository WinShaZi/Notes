# 库其他操作

## 创建库

``` Mysql
# 创建一个库，使其字符集为Charset
CREATE DATABASE Database
CHARSET Charset;
```

## 查看数据库

``` Mysql
# 展示库信息
SHOW CREATE DATABASE Database;
```

## 删除数据库

``` Mysql
# 删除数据库
DROP DATABASE Database;
```

## 修改数据库

``` Mysql
# 修改库Database的字符集
ALTER DATABASE Database CHARSET Charset;
```
