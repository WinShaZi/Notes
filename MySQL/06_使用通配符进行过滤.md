# 使用通配符进行过滤

## %

``` Mysql
# %通配符表示任意字符出现任意次
SELECT Column
FROM Table
WHERE Column LIKE 'Str%';
```

## -

``` Mysql
# _通配符表示任意字符出现一次
SELECT Column
FROM Table
WHERE Column LIKE 'Str_';
```

## ESCAPE

``` Mysql
# 在字符n之后的所有通配符仅作为字符
SELECT Column
FROM Table
WHERE Column LIKE 'Str_n%' escape 'n';
```
