# 输入输出重定向

## 重定向

``` shell
# 将输出重定向到 FILE
command > FILE

# 将输入重定向到 FILE
command < FILE

# 将输出以追加的方式重定向到 FILE
command >> FILE

# 将文件描述符为 n 的文件重定向到 FILE
n > FILE

# 将文件描述符为 n 的文件以追加的方式重定向到 FILE
n >> FILE

# 将输出文件 m 和 n 合并
n >& m

# 将输入文件 m 和 n 合并
n <& m

# 将开始标记 tag 和结束标记 tag 之间的内容作为输入
COMMAND << tag
    # body
tag
```

## 文件描述符

``` shell
# 标准输入
0

# 标准输出
1

# 标准错误输出
2
```
